SportStock Pro 🏀👟
Sistema de Gestión de Inventario para Almacenes Deportivos

SportStock Pro es una solución integral diseñada para optimizar el control de inventario, ventas y administración de usuarios en tiendas de artículos deportivos. Este proyecto fue desarrollado como parte del programa académico en la Fundación Escuela Tecnológica (FET).

🚀 Características Principales
Gestión de Inventario (CRUD): Control total sobre productos, categorías y stock en tiempo real.

Panel Administrativo (Dashboard): Interfaz intuitiva para la visualización de métricas y gestión de datos.

Autenticación de Usuarios: Sistema de acceso seguro con validaciones de formato corporativo y roles administrativos.

Arquitectura Desacoplada: Backend robusto desarrollado en Java y Frontend dinámico basado en estándares modernos de la web.

🛠️ Tecnologías Utilizadas
Backend (API REST)
Java & Spring Boot: Núcleo del servidor y lógica de negocio.

Maven: Gestión de dependencias y construcción del proyecto.

Postman: Testing y documentación de endpoints de la API.

Frontend
HTML5 & CSS3: Estructura semántica y diseño visual minimalista.

JavaScript (Vanilla): Lógica dinámica del lado del cliente y comunicación asíncrona con la API.

📂 Estructura del Proyecto
/src: Código fuente del Backend en Java.

/CSS: Estilos personalizados para la interfaz.

/assets: Recursos gráficos y logotipos de la marca.

index.html: Punto de entrada principal (Login).

dashboard.html: Panel principal de administración.
